# Wikilinks and tags

Two lightweight conventions turn a pile of notes into a connected workspace.

## Wikilinks

Write `[[Note name]]` to link to another note. For example, this links back to [[getting-started]] and over to [[markdown-basics]].

Switch to the **Graph** tab in the sidebar to see every note as a node, with a line drawn for each wikilink. Click a node to open that note.

<Callout type="info" title="How matching works">
A wikilink matches a note by its path without the extension. For a note inside a section, link to it by its full path, like projects/autonote-roadmap.
</Callout>

So to reach the note in the projects section, link to [[projects/autonote-roadmap]].

## Tags

Write a `#tag` anywhere in a note. Neuron collects every tag into the filter at the bottom of the sidebar — click one to narrow the list to notes that mention it.

This note carries a couple:

#guide #links #organization
